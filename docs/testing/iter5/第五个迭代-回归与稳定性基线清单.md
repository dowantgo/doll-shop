# 第五个迭代回归与稳定性基线清单

## 一、功能回归

- 登录 / 注册 / 忘记密码
- 图片验证码 / 邮箱验证码
- 商品列表 / 商品详情
- 购物车
- 普通下单
- 支付
- 秒杀
- 优惠券
- 退款
- 物流
- 后台管理

## 二、稳定性专项

- 登录失败锁定
- 图片验证码频率限制
- 邮箱验证码冷却与频率限制
- 同一用户并发加购
- 同一购物车重复下单
- 订单过期异步清理
- 支付状态轮询限频与缓存命中
- refresh token 自动续签

## 三、关键指标

- 登录限流命中数
- 验证码限流命中数
- 邮箱验证码冷却命中数
- 下单幂等命中数
- 订单过期清理修复数
- 支付状态查询缓存命中数
- 第三方支付查询次数
- 支付状态查询限频命中数
