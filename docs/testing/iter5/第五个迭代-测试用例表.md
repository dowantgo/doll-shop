# 第五个迭代测试用例表

## 范围

- 用户入口限流
- 购物车并发更新
- 普通下单幂等
- 订单过期异步清理
- 支付轮询减压
- Refresh Token 续签

| TC-I5-005 | 购物车 | 更新数量超过库存 | 返回统一错误 | 待测 |
